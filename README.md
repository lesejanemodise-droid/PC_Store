# **PC_Store**
*Designing a Data Warehouse from CSV Raw Data Using Snowflake Schema*
# **Overview**
*This project demonstrates how to design and implement a data warehouse using raw CSV files as the data source and a Snowflake Schema as the dimensional model. The goal is to transform unstructured or semi-structured flat-file data into a structured analytical database optimized for reporting, dashboards, and business intelligence.The warehouse organizes data into fact and dimension tables, where dimensions are normalized into related sub-dimension tables to reduce redundancy and improve data consistency.*
# **Project Objectives**
Import raw CSV data into a staging environment.

Clean and standardize inconsistent source data.

Design a Snowflake Schema using draw.io.

Build dimension and fact tables using SQL.

Load transformed data into the warehouse.
